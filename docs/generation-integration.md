# 生成集成模块

## 核心功能

1. **智能查询路由**：根据用户查询自动判断是列表查询、详细查询还是一般查询，选择最适合的生成策略
2. **查询重写优化**：对模糊不清的查询进行智能重写，提升检索效果
3. **多模式生成**：列表模式 / 详细模式 / 基础模式

## 查询类型

| 类型 | 示例 | 生成策略 |
|------|------|---------|
| list | "推荐几个简单的素菜" | 返回简洁的菜品列表 |
| detail | "宫保鸡丁怎么做" | 提供分步骤的详细指导 |
| general | "做菜需要什么工具" | 提供常规回答 |

## 查询重写

将模糊查询优化为更精确的检索词：

- "做菜" → "简单易做的家常菜谱"
- "好吃的" → "受欢迎的家常菜"

## 多模式生成

### 列表模式
适用于推荐类查询，返回简洁的菜品列表：

```
1. 西红柿炒鸡蛋 - 简单快手，5分钟上桌
2. 炒青菜 - 清爽健康，新手友好
3. 鸡蛋花 - 汤品，营养丰富
```

### 详细模式
适用于制作类查询，提供分步骤指导：

```
## 宫保鸡丁

### 食材
- 鸡胸肉 300g
- 花生米 50g
- ...

### 步骤
1. 鸡胸肉切丁，加料酒、生抽腌制15分钟
2. ...
```

### 基础模式
适用于一般性问题，提供常规回答。
